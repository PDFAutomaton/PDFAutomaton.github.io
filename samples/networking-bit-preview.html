Startup Programs বন্ধ করুন যেগুলো আপনি চালু করেননি

Keyboard-এ Ctrl + Shift + Esc চাপুন — Task Manager খুলবে। উপরে 'Startup' tab-এ click করুন। দেখবেন অনেক program আছে যেগুলো computer চালু হওয়ার সাথে সাথে নিজেই চালু হয় — Spotify, Teams, OneDrive, Skype সহ অনেক কিছু। যেগুলো আপনি daily কাজে use করেন না সেগুলোতে right-click করে 'Disable' করুন। এই একটা কাজেই computer startup time ৩০-৫০% কমে যাবে।

2
Temporary Files পরিষ্কার করুন — এটা বেশিরভাগ মানুষ কখনো করেননি

Keyboard-এ Windows key + R চাপুন। একটা small box আসবে। সেখানে লিখুন %temp% এবং Enter চাপুন। একটা folder খুলবে যেখানে হয়তো হাজার হাজার ফাইল জমে আছে। Ctrl + A দিয়ে সব select করুন, তারপর Delete চাপুন। যদি কিছু ফাইল delete না হয় সেগুলো skip করুন — কোনো সমস্যা নেই। এই জায়গায় GB-তে space জমে থাকে যা computer কে slow করে।

3
এখনই একটা simple maintenance routine তৈরি করুন যাতে এটা আর না হয়
